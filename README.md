#Dashboard de anúncio de carros

O projeto apresenta uma análise de dados de anúncio de carros e um app web desenvolvido com Streamlit.

##Funcionalidades desse app:

-Exibe histograma da quilometragem dos veículos
-Exibe gráfico de dispersão entre KM x preço
-Permite interação com os botões

##Techs utilizadas:

-Python
-pandas
-plotly-express
-streamlit
